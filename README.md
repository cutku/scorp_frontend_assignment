# scorp_app (scorp_frontend)
[Live](https://scorp-frontend-assigment.vercel.app/#/)

## Install the dependencies

```bash
npm install
```

### Start the app in development mode (hot-code reloading, error reporting, etc.)

```bash
npm run dev
```

### Lint the files

```bash
npm run lint
```

### Build the app for production

```bash
npm run build
```
